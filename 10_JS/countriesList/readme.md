### project. Страны в выпадающем списке

Завершите функцию `getDropdown`, чтобы она возвращала HTML-код, который заполнит тег `<select></select>`:

- в качестве дочернего элемента должен быть тег `<option value="">Please select</option>` для каждой страны
- значение должно выводиться в нижнем регистре. Например:
`<option value="netherlands">Netherlands</option>`
